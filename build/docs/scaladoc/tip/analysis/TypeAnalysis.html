tip.analysis

TypeAnalysis

case class TypeAnalysis(program: AProgram) extends DepthFirstAstVisitor[Null] with Product with Serializable

The analysis associates with each node in the AST a tip.types.TipType. The associated term is meant to represent a TipType (when the inference has established a definite type), a TipType with open variables (when the inference has not inferred the identity of all the types appearing as subterms) and a rational recursive TipType in the case of tip.types.Mu
